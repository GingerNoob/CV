# CV Website Repository

This repository contains my personal CV website with a terminal-inspired design. The website is built using HTML, CSS, and JavaScript for a responsive and interactive experience.

## Version

- **CV-Terminal-Style-B&W**: A black-and-white terminal-style interface for a minimalist look.

## Usage

1. Clone the repository: `git clone https://github.com/yourusername/CV.git`
2. Navigate to the version folder: `cd CV-Website/CV-Terminal-Style-B&W`
3. Open `index.html` in a web browser to view the CV.

## Technologies

- HTML5
- CSS3
- JavaScript (ES6+)

## Customization

Edit the `index.html`, `style.css`, and `script.js` files to personalize the content and styling.
